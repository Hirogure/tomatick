# 🍅 Tomatick

**Online Pomodoro Timer for Focus & Productivity**

> tomatick.app — Free, no sign-up required.

## Overview

Tomatick is a beautiful, minimal Pomodoro timer web app targeting English-speaking students and professionals. Built with Next.js + Tailwind CSS, deployed on Vercel.

## Tech Stack

- **Framework**: Next.js (App Router) + TypeScript
- **Styling**: Tailwind CSS
- **Storage**: localStorage (no backend)
- **Hosting**: Vercel (free)
- **Domain**: tomatick.app

## Features (MVP)

- ⏱ 25/5/15 minute Pomodoro cycles
- 🎨 Background color changes per mode (Focus / Short Break / Long Break)
- ✅ Task list with localStorage persistence
- ⌨️ Keyboard shortcut (Space to start/pause)
- 🔔 Desktop notifications
- 📱 Responsive design
- ⚙️ Customizable timer durations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
